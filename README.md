# Argonautes-app

## How to use this App ? 

### 1- clone this project 
### 2- cd backend 
### 3- npm install
### 4- npm run dev

The server should run on localhost port 4000

### 5- cd ../frontend
### 6- npm install
### 7- npm run start

The server should run on localhost port 3000

#### You can use this application right away.
